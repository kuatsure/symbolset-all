# Symbolset All

> Grabs all components for the Symbolset libraries.

[![Bower Version](http://img.shields.io/bower/v/symbolset-all.svg)](https://github.com/kuatsure/symbolset-all)

## Install

```bash
$ bower install symbolset-all -S
```
